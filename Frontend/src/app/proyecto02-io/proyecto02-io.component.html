Problema de la mochila

El modo 0/1 hace que el algoritmo tenga que seleccionar 0 o 1 objetos posibles de cada uno para entrar en la mochila

El modo bounded permite usar una cantidad limitante por cada objeto, de no darse valor se considera infinita

El modo unbounded, se considera que la cantidad posible de cada objeto es infinita, por lo tanto, se encarga de maximizar con base a ésto
